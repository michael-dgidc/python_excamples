# Python Learning Examples

A beginner-friendly Python repo for learning the core building blocks:

- Data types
- Data structures
- Functions
- Control flow

The examples are small, readable, and runnable from the terminal.

## Project Layout

```text
python_learning/
  examples/
    01_data_types.py
    02_data_structures.py
    03_functions.py
    04_control_flow.py
  exercises/
    practice.py
  tests/
    test_practice.py
```

## Getting Started

You need Python 3.10 or newer.

Check Python:

```bash
python --version
```

Run an example:

```bash
python examples/01_data_types.py
```

Run all examples:

```bash
python examples/01_data_types.py
python examples/02_data_structures.py
python examples/03_functions.py
python examples/04_control_flow.py
```

Run tests:

```bash
python -m unittest discover tests
```

## Learning Path

1. Start with `examples/01_data_types.py`.
2. Move to `examples/02_data_structures.py`.
3. Learn reusable code in `examples/03_functions.py`.
4. Practice decisions and loops in `examples/04_control_flow.py`.
5. Complete the tasks in `exercises/practice.py`.
